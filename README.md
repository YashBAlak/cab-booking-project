# 🚖 Cab Booking Application – Salesforce

This project is a Salesforce-based Cab Booking System built using Apex, Lightning Web Components (LWC), and Flow Automation.

## 🔹 Features
- Customer registration and login
- Cab reservation with multi-step booking wizard
- Automated email notifications to customers and drivers
- Validation rules and field dependencies for booking data
- Secure backend authentication using Apex

## 🛠️ Technologies Used
- Salesforce Apex
- Lightning Web Components (LWC)
- SOQL
- Salesforce Flow
- HTML, CSS, JavaScript

## 📂 Project Structure
cabBookingApp/
 ├── lwc/
 ├── classes/
 ├── objects/
 └── flows/

## 🚀 How It Works
1. User signs up or logs in
2. User selects pickup and drop location
3. System calculates pricing and assigns driver
4. Booking confirmation email is sent automatically
